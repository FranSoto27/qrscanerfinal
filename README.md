Escáner QR

Aplicación móvil desarrollada con Angular, Ionic y Capacitor que permite escanear códigos QR directamente desde la cámara del celular.

¿Qué hace?


Accede a la cámara del dispositivo para detectar y leer códigos QR en tiempo real.
Pensada para ejecutarse como app móvil híbrida (Android/iOS) a partir de una misma base de código web.


Tecnologías


Angular
Ionic Framework
Capacitor (acceso nativo a cámara)
TypeScript


Aprendizajes

Acá aprendí a trabajar con desarrollo híbrido: una sola base de código en Angular que se empaqueta como app nativa con Capacitor, y el manejo de permisos y APIs nativas del dispositivo (cámara) desde una app web.
